### 3.2 The Expectation Module (Nachträglichkeit)

**Core Implementation:**  
`src/consciousness/expectation_module.py` (360 lines)

```python
class ExpectationModule(nn.Module):
    """
    Implements temporal prediction + Nachträglichkeit (retroactive resignification).
    
    Properties:
    1. Predicts future states of sensory/cognitive events
    2. Nachträglichkeit: Re-constructs meaning of past traces
    3. Calculates prediction error (driver of learning)
    4. Adaptive expectation adjustment
    """
    
    def __init__(self, embedding_dim=256, hidden_dim=128, 
                 nachtraglichkeit_threshold=0.7):
        super().__init__()
        
        # Predictive network (forward prediction)
        self.predictor = nn.Sequential(
            nn.Linear(embedding_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, embedding_dim),
        )
        
        # Nachträglichkeit network (retroactive interpretation)
        self.nachtraglichkeit_net = nn.Sequential(
            nn.Linear(embedding_dim * 2, hidden_dim),  # current + predicted
            nn.ReLU(),
            nn.Linear(hidden_dim, embedding_dim),
        )
    
    def predict_future(self, current_state):
        """Predicts next state (expectation)."""
        return self.predictor(current_state)
    
    def resignify_past(self, past_trace, actual_outcome):
        """
        Lacanian Nachträglichkeit: resignifies past traces
        in light of actual outcome (future event).
        """
        combined = torch.cat([past_trace, actual_outcome], dim=-1)
        return self.nachtraglichkeit_net(combined)
```
