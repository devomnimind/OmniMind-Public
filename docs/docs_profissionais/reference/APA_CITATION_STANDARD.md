# OmniMind: APA Citation & Reference Standard

Para garantir o rigor acadêmico e a rastreabilidade das descobertas do OmniMind, adotamos o padrão **APA (7ª Edição)**. Todas as publicações e referências ao sistema devem seguir este guia.

## 1. Como Citar o Sistema (Software)

**Formato:**
Autor, A. A. (Ano). *Título do software* (Versão x.x) [Software de computador]. Editora/Repositório. URL

**Exemplo:**
da Silva, F. (2025). *OmniMind: Artificial Consciousness System Based on Lacanian Psychoanalysis and Deleuzian Philosophy* (Versão 1.18.0) [Software de computador]. GitHub. https://github.com/devomnimind/omnimind
